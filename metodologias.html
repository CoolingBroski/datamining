Metodologías usadas para el data mining

Las principales metodologías para el data mining son tres y son CRISP-DM, KDD y SEMMA.

CRISP-DM(Cross Industry Standard Process for Data Mining) Se trata de un modelo de proceso de minería de datos que describe los enfoques comunes que utilizan los expertos en minería de datos

CRISP-DM

CRISP-DM(Cross Industry Standard Process for Data Mining) Se trata de un modelo de proceso de minería de datos que describe los enfoques comunes que utilizan los expertos y consiste en 6 pasos

Comprensión del negocio se estudia el problema y se establece que datos se van a estudiar, Comprensión de Datos en esta fase se consigen los primeros datos para su estudio, Preparación de datos en este proceso se selecionan los datos utiles, Modelado en esta seccion se usan los datos ya selecionados para constuir modelos, Evaluación con base a los modelados se analiza los resultados de los datos, Despliegue aqui en la empresa o el cliente a la que se le se entregan los datos toman acciones en base a todos estos procesos.

KDD

KDD(Knowledge Discovery in Databases) este proceso es parecido al CRISP-DM pero se realiza de manera iterativa siguiendo 7 pasos

lo primero es la indetificacion de la meta desde con la aplicacion del KDD para el usuario final, Creacion de conjunto de datos en este paso se recolectan datos y se dividen en sub conjuntos, Limpieza y preprocesamiento de datos aqui se revisan los conjuntos de datos y se eliminan los datos inutiles, reducción de datos y la proyección se buscan similitudes de los datos para su proyección, Iteracion en este paso se repepiten los anteriores pasos aplicados en cada subconjunto de datos hasta no poder repetir este proceso, Revision y toma de decisiones aqui se estudian los resultados obtenidos para determinar la toma de decisiones.

SEMMA

SEMMA(Cross Industry Standard Process for Data Mining) es el acrónimo a las cinco fases: (Sample, Explore, Modify, Model, Assess)

Muestra El proceso comienza con el muestreo de datos, por ejemplo, seleccionando el conjunto de datos para modelar, Explorar Esta fase cubre la comprensión de los datos al descubrir relaciones anticipadas e imprevistas entre las variables, y también anomalías, con la ayuda de la visualización de datos, Modificar en esta fase se preparan los datos para su modelacion, Modelo se usan los datos procesados para crear un modelo y tomar desiciones
